## Getting Started

To get started, clone this repository in your terminal.

### Installing

To get this project running on your local machine, run npm install and npm start.

```
npm install
npm run start
```

The following packages will also need to be installed:

### Packages Used

React
React-Router
Moment
Node-Sass
FortAwesome
